# Vroom-Africa-
Ce projet est inspiré d’une mini F1, intégrant un microcontrôleur central sur PCB multicouche, batterie basse pour stabilité, deux moteurs DC arrière pour la propulsion et un servomoteur avant pour la direction. Carrosserie amovible aérodynamique, conçue pour enseigner l’électronique, la mécanique et les bases de l’ingénierie automobile.
